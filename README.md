# Escopo do projeto

Projeto de portifólio/currículo para apresentar minhas habilidades, experiências, formação e etc.

O projeto inclui:
(a) VueJs

## Iniciando...

- `git clone https://github.com/guilherme-studier/guillherme-studier-curriculum`
- `cd guillherme-studier-curriculum`

### Instalar pacotes do node

> _Objetivo: Necessário ter npm instalado

- `npm update`
- `npm run dev`
- `npm run build`
